Join us for an immersive workshop that will introduce you to the world of Terraform, an open-source Infrastructure as Code (IaC) tool. In this session, we will delve into the core concepts of Terraform, including Resources, Data Sources, Modules, and Providers, and explore how it seamlessly integrates with Amazon Web Services (AWS). Don't worry if you're new to AWS; we'll provide a high-level overview to get you up to speed.

Throughout the workshop, you will gain a deep understanding of the benefits of Infrastructure as Code and discover how Terraform can revolutionize your deployment workflows. To put your knowledge into practice, we have prepared an exciting hands-on lab where you'll have the opportunity to deploy a Wordpress site to AWS using Terraform.
By the end of this workshop, you will have achieved the following learning goals:

- Understand the ways you can deploy workloads to AWS – Manually vs Programmatically  
- Understand what Infrastructure as Code (IaC) is, and why it is beneficial  
- Understand what Terraform is and how to use it on AWS  
- Understand the basic concepts of Terraform such as Providers, Resources, Data Sources, Modules, and Terraform State  
- Apply this knowledge to deploy a Wordpress site to AWS with Terraform
