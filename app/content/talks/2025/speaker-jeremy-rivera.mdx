## Details
![](/images/talks/2025/speaker-jeremy-rivera/1749730431912.webp)

Accessibility is a crucial part of building inclusive web applications, but it typically takes a back seat during development. In this session, you’ll learn how to identify common accessibility pitfalls and implement practical strategies to overcome them. Discover tools like `axe DevTools` and `eslint-plugin-jsx-a11y`, explore best practices for keyboard navigation, focus management, and ARIA roles, and see how accessible component libraries can simplify your work. By the end, you’ll have insights to build web apps that are not only functional but truly accessible, so everyone can enjoy them!

[Deque publication: Building accessible apps with Next.js and axe DevTools](https://www.deque.com/blog/building-accessible-apps-with-next-js-and-axe-devtools/)
