## Details
Join us for an engaging and informative workshop where you'll dive deep into the world of AWS, Terraform, and Kubernetes. Led by Cindy Singleton, this workshop is designed for those eager to learn about cloud infrastructure and automation.

* **Understanding AWS, Terraform, and Kubernetes:** Discover the basics of these powerful tools and how they integrate seamlessly.
* **Deploying an EKS Cluster via Command Line:** Learn the why and how of deploying a Kubernetes cluster in AWS. Understand the importance of repeatability, quality, and speed in deployments.

Hands-On Tasks:
  * **Terraform Installation & Setup:** Install Terraform and learn how to configure your provider.tf (or aws.tf) with the necessary credentials.
  * **Step-by-Step Terraform Guide:** Write `.tf` files to deploy every component of an Amazon EKS cluster, including the VPC, subnets, IAM roles and policies, the EKS control plane, Node Groups, Nginx, and Load Balancers.
  * **Launching Your Website:** Deploy your website on EKS and access it live online.
  * **Cleanup:** Conclude the workshop by safely tearing down your infrastructure using the terraform destroy command.

**Who is this for?**
This workshop is perfect for anyone looking to gain hands-on experience and a deeper understanding of cloud services and orchestration tools. Whether you're a beginner or already have some familiarity with AWS and Kubernetes, you'll find this session both fun and enriching.
