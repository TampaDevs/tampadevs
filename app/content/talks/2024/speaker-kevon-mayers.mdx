## Details
Join us in an immersive Terraform CI/CD and Testing on AWS workshop hosted by St. Petersburg College, Tampa Bay Techies, Tampa Devs, and Java User Group.

In this workshop, you will learn how to leverage the new **[Terraform Testing Framework](https://www.hashicorp.com/blog/testing-hashicorp-terraform)** introduced in \`v1.6.0\` to perform manual unit, integration, and security tests for your Terraform projects. You will then learn how to build a basic Terraform CI/CD pipeline on AWS to automate these tests and deploy your infrastructure safely and securely.
