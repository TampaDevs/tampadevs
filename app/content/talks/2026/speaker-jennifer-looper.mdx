## Details
In this hands-on workshop, you'll boost your "Media IQ" - that is, you'll learn about modern image formats (WebP, AVIF), optimization techniques that actually matter, using Cloudinary for on-the-fly transformations and delivery, and how your images impact Core Web Vitals. You'll build hands-on skills through a real Next.js e-commerce project.

![](/images/talks/2026/speaker-jenn-looper/highres_533290049.webp)
